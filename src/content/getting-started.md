# Getting Started

Follow these steps to get BeSimple CodeEditor running on your machine for development.

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm (usually comes with Node.js)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-repo/besimple-codeeditor.git
   ```

2. Navigate to the app directory:
   ```bash
   cd BeSimple - CodeEditor/app
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

Start the development environment:
```bash
npm start
```

### Building for Production

To create a distributable version of the app:
```bash
npm run build
```
