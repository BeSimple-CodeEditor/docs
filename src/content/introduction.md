# Welcome to BeSimple
> The distraction-free, high-performance coding environment.

BeSimple CodeEditor is built with **Electron** and **Monaco**, focusing on simplicity while providing powerful features like real-time folder watching, custom themes, and intelligent file management.

### Why BeSimple?

- **Minimalist Design**: We stripped away the bloat so you can focus on your code.
- **High Performance**: Built on top of the same engine as VS Code, but lighter.
- **Extensible**: Easily add your own themes and configurations.
